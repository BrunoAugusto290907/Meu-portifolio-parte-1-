# Meu Portfólio Pessoal 🚀

Um portfólio simples e direto desenvolvido como atividade prática para o curso de Análise e Desenvolvimento de Sistemas (ADS). O objetivo deste projeto foi consolidar os fundamentos de estruturação web e aplicar estilos básicos.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação de todo o conteúdo, textos e imagens.
* **CSS3:** Estilização visual, cores de fundo e alinhamento.

## 📚 Minhas Habilidades e Estudos Atuais
Estou sempre em busca de aprender novas ferramentas para criar soluções criativas e eficientes. No momento, meu foco é:
* **Estrutura e Estilo:** HTML e CSS
* **Linguagens e Dados:** Python, MySQL, JavaScript e Java
* **Idiomas:** Inglês (em estudo contínuo focando em regras gramaticais e leitura)

## 🚀 Como executar o projeto
Como este é um projeto estático, não é necessária nenhuma instalação complexa. 
1. Faça o clone ou o download deste repositório.
2. Extraia os arquivos.
3. Dê um duplo clique no arquivo `index.html` para abri-lo em qualquer navegador de sua preferência.
